# ゲームパッド・バウンスゲーム 計画書

## 概要
Three.js を使用した2Dバウンスゲーム。プレイヤーはボールとなり、四角い枠の中でジャンプ・バウンドしながら粒状の敵を全て消すことでクリアとなる。

## 技術仕様
- **描画**: Three.js (2D的な表現をOrthographicCameraで実現)
- **入力**: Gamepad API (左スティック/十字キーで方向、Aボタンでジャンプ)
- **構成**: 単一 index.html ファイル

## ゲーム仕様

### プレイヤー
- ボール型（円形スプライト）
- マリオ風のジャンプ（重力あり）
- 壁に当たると跳ね返る
- ゲームパッドの左スティック/十字キーで左右移動、ボタンでジャンプ

### フィールド
- 四角い枠（壁）で囲まれた領域
- 上下左右の壁で跳ね返る

### 敵
- 小さな粒状の敵がフィールド内にランダム配置
- プレイヤーがぶつかると消える

### クリア条件
- 全ての敵を消したらゲームクリア
