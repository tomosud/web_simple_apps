# バウンスゲーム

Three.js で作った2Dバウンスゲームです。ゲームパッド対応。

## 遊び方

1. `run.bat` をダブルクリックしてサーバーを起動
2. ブラウザで `http://localhost:8089` が開きます

## 操作方法

| 操作 | ゲームパッド | キーボード |
|------|------------|-----------|
| 左右移動 | 左スティック / 十字キー | ← → / A D |
| ジャンプ | A ボタン | スペース / ↑ / W |
| ダッシュ | B / X / RB ボタン | Shift / Z |
| リスタート | START / A ボタン | Enter |

## ルール

- プレイヤー（緑のボール）を操作して、フィールド内のカラフルな敵（小さな丸）を全て倒すとクリア
- 敵はフィールド内を自由に動き回り、壁で跳ね返ります
- 1体倒すと **100点**

## Bダッシュ & ジャンプコンボ

- **ダッシュボタン**を押しながら移動すると高速移動（Bダッシュ）
- ダッシュ中にジャンプすると **大ジャンプ**
- 着地直後にすぐジャンプすると **コンボ** が発生し、さらに高くジャンプ（最大5コンボ）

## 軌跡システム

- ボールは移動すると **滑らかな尻尾のような軌跡** を残します（CatmullRom曲線）
- **軌跡に触れた敵も倒せます**
- 敵を倒すと **スピードアップ**（1体につき +0.25倍、最大 x4.0）
- スピードが速いほど **軌跡が長く太くなる** → より多くの敵を巻き込める
- 連続で敵を倒すことでスピードが上がり続け、軌跡がどんどん伸びる
- **壁に当たると少し減速**（-0.15倍）
- 時間経過で **徐々に通常速度に戻る**（毎秒 -0.3倍）
- HUDにスピード倍率を表示
